== Description ==

RemitRadar remittance calculator/widget, allows you to provide realtime exchange rates to your visitors. Easy installation and configuration process, let you to start streaming realtime data in seconds. This useful widget, updated regularly and  provides only up to date actual information. Great widget to entertain your audience especially if your content or website is related to immigration, international business or other sectors that are dependant on the international money transfers. If you are webmaster managing website that process over 1,000,000 monthly sessions. Please get in touch with our team to set up your compensation agreement at widgets@remitradar.com

== Installation ==

1. Upload `plugin-folder` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin and add a shortcode or widget to your page
